= Shiva Web - WAR

To use, simply add the following code to your project:

.web(-fragment).xml
[source,xml]
----------

<filter>
	<filter-name>JadeTemplateFilter</filter-name>
	<filter-class>nz.co.aetheric.shiva.web.jade.JadeFilter</filter-class>
</filter>

<filter-mapping>
	<filter-name>JadeTemplateFilter</filter-name>
	<url-pattern>/*</url-pattern>
</filter-mapping>

----------
